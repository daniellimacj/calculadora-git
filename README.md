# calculadora-git
Calculadora em Python para a tarefa do módulo Git

# Como executar o arquivo .sh
Para facilitar a execução do projeto, foi criado um script em shell (.sh). Siga os passos abaixo para utilizá-lo:
- Abra o terminal na pasta do projeto.
- Dê permissão de execução ao arquivo:
  chmod +x calculadora.sh
- Execute o script:
  ./calculadora.sh
Esse script automatiza a execução do programa em Python, evitando a necessidade de digitar o comando manualmente toda vez.

# Explicação do código Python (Calculadora)
O código da calculadora foi desenvolvido em Python com o objetivo de realizar operações matemáticas básicas de forma simples e interativa.

## Estrutura do programa
O programa segue uma sequência lógica:
1. Entrada de dados
   -  O usuário informa dois números.
   -  Também escolhe qual operação deseja realizar.
2. Operações disponíveis
   - Soma
   - Subtração
   - Divisão
   - Multiplicação
3. Processamento
   - O programa utiliza estruturas condicionais (if, elif, else) para identificar qual operação foi escolhida.
4. Saída
   - O resultado da operação é exibido no terminal.
